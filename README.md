# SOAR Pypeit PASSTA

Wrapper code to run pypeit for SOAR/Goodman data

This is a supplementary repository (still under construction) to run pypeit on SOAR data from PASSTA program (PI: Hosseinzadeh) for quicklook and rapid classification. For an example on how to run pypeit for SOAR please look here: https://github.com/charliekilpatrick/soar_pypeit_example

1) An account on LCO observing portal. If you don't have one, please register here: https://observe.lco.global/accounts/register
Once that is done you can store your account credentials as environment variables (LCOGT_USERNAME, LCOGT_PASSWORD) that the querying script uses.
2) Access to the PASSTA proposal on LCO portal, so you can submit observations & data can be downloaded on your machine using the script. Please contact @Griffin Hosseinzadeh with your email (that you used for registration) and he can add you.
